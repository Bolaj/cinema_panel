# SAIL Assignment

This Node.js application includes three modules:
1. **Cinema Panel**: Manage log to fetch all cinemas, fetch a single cinema.
2. **Library Module**: Track book stock and monitor checked-out books.
3. **NGO Expense Tracker**: Track expenses related to orphanage visits and manage items delivered to orphanages.

---

## Features

### **1. Cinema Panele**
- Add new cinema.
- Fetch all cinemas or search for a specific cinema.
- Update and delete Cinema.

### **2. Library Module**
- Add books to the library with stock and checkout tracking.
- Fetch all books or search for a specific book.
- Track how many books are checked out.

### **3. NGO Expense Tracker**
- Add orphanages and track visits.
- Log expenses for each orphanage.
- Track total spending and manage items delivered.

---

## Technologies Used

- **Node.js** with **Express.js** for backend development.
- **MongoDB** with **Mongoose** for data storage.
- **Insomnia** for API testing.
